# Campus AI Simulation Backend

This repository contains the Flask backend for an AI-driven campus multi-agent simulation. The backend powers intelligent student agents that can answer natural-language questions, reason about their current location, detect nearby agents, explain schedule deviations, evaluate user commands, and make decision-based responses.

The system integrates several AI methods from the course project, including neural NLP intent classification, word representations, attention-based sequence modeling, expected utility, value of information, EM hidden activity-mode inference, Q-learning reinforcement learning, and multi-agent congestion reasoning.

The backend communicates with a React frontend and returns structured AI evidence such as predicted intent, confidence, attention weights, word representation evidence, EM mode probabilities, Q-learning recommendations, and campus reasoning results.
